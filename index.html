LIMBS

A modern, minimal css boilerplate built off Skeleton
Designed to be a starting point, not a library.
GRID

Like Skeleton, the grid is a 12 column grid with a max width of 960px. It shrinks at smaller sizes. columns could be nested, but that can result in tiny divs on mobile devices.

<!-- columns should be the immediate child of a .row -->
<div class="row">
  <div class="one column">One</div>
  <div class="eleven columns">Eleven</div>
</div>

<!-- just use a number and class 'column' or 'columns' -->
<div class="row">
  <div class="two columns">Two</div>
  <div class="ten columns">Ten</div>
</div>

<!-- there are a few shorthand columns widths as well -->
<div class="row">
  <div class="one-third column">1/3</div>
  <div class="two-thirds column">2/3</div>
</div>

  <div class="one-half column">1/2</div>
