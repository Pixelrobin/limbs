ABOUT

Limbs was created to extend upon Skeleton and provide a little more functionality (hence the name). It was built to be extended, and not reccommended to be used on it's own. Things like the default font and primary buttons were removed in favor of allowing more customizability. Several changes were made to include compatibility with high contrast views.

Limbs consists of 3 files:

  • normalize.css by Anish Athalye to reset styles
  • limbs.css is the main css file
  • limbs.js Adds javascript functionality such as toggleable menus and high contrast detection
SIZING

Pretty much all the sizing in Limbs is based on rem, so it can change on different screen sizes.

1rem == 16px on normal desktop screens.

1rem === 24px on screens 2560+ px wide

Heading tags get a little smaller on mobile screens.

BUTTONS

Limbs includes two types of buttons: a normal one and an icon variety. Buttons are 2rem (32px) high and icon buttons are 2x2rem (32x32px). It is up to you to decide what to put in them. Also, margin and padding in icon buttons are up to you to implement based on your icon size. This page uses the feather icon pack.

FORMS

Forms in limbs don't differ much from Skeleton other than in style. Combine them with the grid to create more interesting layouts.
